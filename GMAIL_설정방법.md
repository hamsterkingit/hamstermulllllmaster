# 메일이 안 갈 때 — Gmail 설정 (코딩 몰라도 됨)

"Missing credentials" 또는 "Gmail 설정이 없어요" 라고 나오면, 아래만 따라 하면 됩니다.

---

## A. 인터넷(Vercel) 주소로 쓰고 있을 때

1. 브라우저에서 **vercel.com** 들어가서 로그인한다.
2. 내가 만든 **프로젝트 이름**을 클릭한다.
3. 위쪽 메뉴에서 **Settings** 를 클릭한다.
4. 왼쪽에서 **Environment Variables** 를 클릭한다.
5. **Add** 또는 **Add New** 버튼을 누른다.
6. **첫 번째 변수**
   - Name 칸에: `GMAIL_USER`
   - Value 칸에: 메일 보낼 때 쓰는 Gmail 주소 (예: dlrkdfla2@gmail.com)
   - Save 누른다.
7. 다시 **Add** 해서 **두 번째 변수**
   - Name 칸에: `GMAIL_APP_PASSWORD`
   - Value 칸에: Gmail 앱 비밀번호 16자 (공백 없이, 예: rcqvtnaholhbdsd)
   - Save 누른다.
8. 위쪽 메뉴에서 **Deployments** 로 간다.
9. 맨 위에 있는 배포 줄 오른쪽 **점 세 개(⋯)** 를 누른다.
10. **Redeploy** 를 누르고, 끝날 때까지 기다린다.

→ 이렇게 하면 메일 보내기가 됩니다. (앱 비밀번호는 Google 계정 보안에서 "앱 비밀번호" 검색해서 발급)

---

## B. 이 컴퓨터에서 "시작하기" 더블클릭해서 쓸 때

1. 이 프로젝트가 들어 있는 **폴더**를 연다.
2. **`.env`** 라는 이름의 파일이 있는지 본다. (숨김 파일일 수 있음)
3. 없으면 **`.env.example`** 파일을 **복사**해서 이름을 **`.env`** 로 바꾼다.
4. **`.env`** 파일을 메모장으로 연다.
5. 다음 두 줄이 있는 곳을 찾아서, 등호 뒤만 본인 걸로 바꾼다.
   - `GMAIL_USER=` 뒤에 → 보낼 Gmail 주소
   - `GMAIL_APP_PASSWORD=` 뒤에 → 앱 비밀번호 16자 (공백 없이)
6. **저장**하고 메모장을 닫는다.
7. "재고발주 서버" 검은 창을 닫았다가, **시작하기**를 다시 더블클릭해서 연다.

---

설정 후에도 같은 오류가 나오면, 변수 이름이 `GMAIL_USER`, `GMAIL_APP_PASSWORD` 인지, 띄어쓰기나 따옴표가 들어가지 않았는지 확인하세요.
